# Day 1 - SQL Basics

## Topics Covered

- CREATE TABLE
- INSERT INTO
- SELECT
- WHERE
- AND
- OR
- NOT
- LIKE
- % Wildcard
- _ Wildcard

## Key Learnings

- WHERE filters rows.
- AND requires all conditions to be true.
- OR requires at least one condition to be true.
- NOT reverses a condition.
- % matches zero or more characters.
- _ matches exactly one character.
